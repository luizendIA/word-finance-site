Integre USDC Solana no seu checkout.

WordFinance Pay e um gateway nao-custodial. Ele cria cobrancas, exibe QR/deep link e notifica sua loja quando o pagamento e confirmado. O dinheiro vai direto para a carteira Solana do lojista.

Baixar plugin WooCommerce Ver health do gateway

URL base

https://wordfinance-gateway-production.up.railway.app
Para WooCommerce, cole esta URL no campo URL do Gateway do plugin.

WooCommerce

1. Instale

Envie a pasta/zip `wordfinance-pay` para o WordPress e ative o plugin.

2. Configure

Em WooCommerce > Pagamentos, cole a URL do gateway e sua carteira Solana.

3. Salve

O plugin registra automaticamente o Merchant ID.

4. Venda

O cliente escolhe Bitcoin & USDC e e redirecionado para a pagina de pagamento.

Registrar lojista

POST /api/gateway/merchant/register
Content-Type: application/json

{
  "name": "Minha loja",
  "solanaWallet": "carteira_solana_do_lojista",
  "webhookUrl": "https://minhaloja.com/wc-api/wf_confirm",
  "platform": "woocommerce",
  "siteUrl": "https://minhaloja.com"
}

Criar cobranca

POST /api/gateway/charge
Content-Type: application/json

{
  "merchantId": "wf_merch_xxx",
  "amountBRL": 99.90,
  "description": "Pedido #123",
  "orderId": "123",
  "customerEmail": "cliente@email.com",
  "webhookUrl": "https://minhaloja.com/wc-api/wf_confirm",
  "returnUrl": "https://minhaloja.com/obrigado"
}

Resposta

{
  "chargeId": "ch_xxx",
  "status": "pending",
  "paymentUrl": "https://wordfinance-gateway.../pay/ch_xxx",
  "deepLink": "wordfinance://pay?...",
  "totalUsdc": 18.54,
  "merchantUsdc": 18.35,
  "feeUsdc": 0.19
}

Webhook enviado para a loja

{
  "chargeId": "ch_xxx",
  "orderId": "123",
  "status": "paid",
  "paymentMethod": "usdc",
  "amountBRL": 99.90,
  "totalUsdc": 18.54,
  "merchantId": "wf_merch_xxx",
  "provider": "wordfinance",
  "txSignature": "assinatura_solana"
}
