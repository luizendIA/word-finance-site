Vantagens

Um gateway feito para quem quer receber stablecoin sem friccao.

A primeira versao foca em USDC Solana porque e rapida, barata e permite modelo nao-custodial. O lojista recebe direto na propria carteira.

Nao-custodial

A WordFinance organiza a cobranca e a confirmacao, mas nao guarda seed, chave privada ou saldo do lojista.

Plugin pronto

WooCommerce ja pode chamar o gateway, registrar lojista, criar cobranca e receber webhook de confirmacao.

API aberta

Qualquer dev pode integrar em SaaS, marketplace, checkout proprio, plataforma de curso ou loja customizada.

API

Integre em qualquer plataforma.

O plugin WooCommerce usa a mesma API publica. Uma loja customizada tambem consegue integrar com poucas chamadas HTTP.

POST /api/gateway/merchant/register
{
  "name": "Minha loja",
  "solanaWallet": "carteira_solana",
  "webhookUrl": "https://loja.com/wf_confirm"
}

POST /api/gateway/charge
{
  "merchantId": "wf_merch_xxx",
  "amountBRL": 99.90,
  "description": "Pedido #123",
  "orderId": "123"
}
Comece

Instale, cole sua carteira Solana e receba USDC.
