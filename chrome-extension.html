Word Finance Wallet no navegador.

A Word Finance Wallet esta publicada na Chrome Web Store e permite usar a carteira em dApps e jogos Web3 no Chrome, mantendo a seed cifrada no dispositivo do usuario.

Instalacao

  1. Use o botao da Chrome Web Store para abrir a pagina oficial.
  2. Clique em Usar no Chrome e confirme a instalacao.
  3. O ZIP v1.3.1 permanece disponivel apenas como alternativa para instalacao manual.

O que ela faz

  • Cria ou importa carteira local nao custodial.
  • Conecta com dApps Solana via provider injetado.
  • Mostra saldos SOL, USDC, PIXC e fluxo Lightning/Pix interno.
  • Pede aprovacao antes de conexoes, assinaturas e transacoes.
  • Protege novas carteiras com PIN de seis digitos e auto-lock.

Seguranca

Guarde sua seed offline. A Word Finance nao recupera seed perdida e nao custodia fundos. A versao 1.3.0 esta publicada oficialmente na Chrome Web Store. Instale somente pelo link oficial acima ou, se souber realizar a instalacao manual, pelo ZIP disponibilizado neste site.
